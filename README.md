#  TrustVault Protocol

Global P2P Escrow & Settlement System.

## Architecture
- **Go**: Orchestration & Logistics APIs
- **Rust**: High-security Cryptographic Validation
- **Solidity**: Non-custodial Smart Contract Vault
- **React/Vite**: Cinematic Glassmorphism UI

## Quick Start
1. Run `make all` to generate Protobuf DNA and build the Rust bridge.
2. Run `docker-compose up` to start the backend and database.
3. In `/src`, run `npm install && npm run dev` for the UI.
